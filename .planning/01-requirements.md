# MEF — 기능 요구사항 (MoSCoW)

> 기준일: 2026-05-04  
> 분류 기준: MVP(과제 제출) 범위 기준으로 Must/Should/Could/Won't 구분

---

## Must Have — 반드시 있어야 한다 (MVP 필수)

### 사용자 인증
- [ ] M-01: 이메일/비밀번호로 회원가입 및 로그인
- [ ] M-02: 로그인 상태 유지 (자동 로그인)

### 시나리오 생성 (ScenarioAgent)
- [ ] M-03: 사용자가 자유 텍스트로 상황을 입력할 수 있다
  - 예: "카페에서 음료를 주문하는 상황", "면접에서 자기소개 하기"
- [ ] M-04: 입력된 상황을 바탕으로 AI가 대화 시나리오(배경, 역할, 목표)를 생성한다
- [ ] M-05: 생성된 시나리오를 사용자가 확인하고 대화를 시작할 수 있다

### 실시간 대화 (ConversationAgent)
- [ ] M-06: 사용자가 텍스트로 AI 파트너에게 메시지를 보낼 수 있다
- [ ] M-07: AI가 시나리오 맥락을 유지하며 자연스럽게 응답한다
- [ ] M-08: 대화 히스토리가 화면에 표시된다 (채팅 UI)
- [ ] M-09: 사용자가 대화를 종료할 수 있다

### 피드백 (FeedbackAgent)
- [ ] M-10: 대화 종료 후 AI가 자동으로 피드백을 생성한다
- [ ] M-11: 피드백은 문법 오류, 표현 개선, 자연스러움 3가지 항목으로 구성된다
- [ ] M-12: 잘못된 문장과 개선된 문장을 나란히 비교해서 보여준다

### 대화 기록
- [ ] M-13: 완료된 대화 세션 목록을 조회할 수 있다
- [ ] M-14: 과거 대화 내용과 피드백을 다시 열람할 수 있다

---

## Should Have — 있으면 좋다 (시간 여유 시 구현)

### 시나리오 생성 강화
- [ ] S-01: 난이도 선택 (초급 / 중급 / 고급)
- [ ] S-02: 카테고리 선택 지원 (일상, 비즈니스, 여행, 학업)
- [ ] S-03: 최근 사용한 시나리오 빠른 재시작

### 대화 경험 개선
- [ ] S-04: 음성 입력 지원 (STT — Speech to Text)
- [ ] S-05: AI 응답을 음성으로 재생 (TTS — Text to Speech)
- [ ] S-06: 힌트 버튼 (이 상황에서 쓸 수 있는 표현 제안)

### 피드백 개선
- [ ] S-07: 전체 대화 점수 (0–100점) 제공
- [ ] S-08: 피드백 항목별 별점 또는 이모지 리액션

### 사용자 경험
- [ ] S-09: 온보딩 튜토리얼 (첫 실행 시 앱 사용 방법 안내)
- [ ] S-10: 다크 모드 지원 (Flutter ThemeData로 구현)

---

## Could Have — 여유가 있다면 (추후 기능)

### 분석 기능 (AnalysisAgent — 유료)
- [ ] C-01: 누적 대화 데이터 기반 약점 패턴 분석
  - 예: "시제 오류가 전체 오류의 42%입니다"
- [ ] C-02: 주간/월간 실력 성장 리포트
- [ ] C-03: 약점 기반 집중 훈련 시나리오 자동 추천

### 소셜/게임화
- [ ] C-04: 연속 학습 스트릭(streak) 표시
- [ ] C-05: 학습 목표 설정 및 달성 알림
- [ ] C-06: 친구에게 시나리오 공유

### 콘텐츠
- [ ] C-07: AI가 생성한 추천 시나리오 라이브러리 제공
- [ ] C-08: 시나리오 즐겨찾기 저장

---

## Won't Have — 이번 버전에서는 구현하지 않는다

- [ ] W-01: 사람 튜터와 매칭 (1:1 화상 연결)
- [ ] W-02: 커뮤니티 게시판 / 소셜 피드
- [ ] W-03: 그룹 대화 (멀티플레이어 롤플레이)
- [ ] W-04: 결제 시스템 실제 연동 (인앱결제 등) — 기획만 반영
- [ ] W-05: iOS/Android 앱스토어 실제 배포
- [ ] W-06: 다국어 지원 (영어 외 언어 학습)
- [ ] W-07: 오프라인 모드 (인터넷 없이 학습)

---

## 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 응답 속도 | AI 응답은 3초 이내 (UX 기준) |
| 플랫폼 | iOS 15+, Android 10+ |
| 네트워크 | Wi-Fi 및 LTE 환경에서 정상 동작 |
| 보안 | API 키는 서버 사이드에서만 사용, 클라이언트에 노출 금지 |
| 언어 | UI는 한국어, 대화 콘텐츠는 영어 |
| UI 일관성 | Flutter 자체 렌더링 엔진으로 iOS/Android 동일한 UI 보장 |
